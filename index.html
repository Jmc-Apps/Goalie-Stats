Cloud JSON database file

Version 2.03 adds PWA install/offline support plus the low-cost cloud-file workflow. Choose a JSON database file saved in iCloud Drive, Google Drive, Dropbox, OneDrive, or another synced folder. The app will keep a local copy for match-day reliability. On desktop browsers that support direct file access it can save back to the selected file; on iPhone it will create/share a JSON backup file when the match outcome is saved.
